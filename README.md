# How to use

Install git via https://git-scm.com/downloads
use Git Bash when right click
input text : git clone "input repo github"
-----------------------------------------------
when update all use command : git add .
git add . => git add all
when want to cancel file add use command : git reset .
git reset . => reset file add all
after git add use command : git commit -m "Input text here"
after git commit use command : git push origin master
origin "" => "branch to push"
